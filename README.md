# ClientSync - Strategic Account Manager

A professional, bright, and responsive data-entry tool designed for strategic account management. Built with **React 19**, **Vite 8**, and **Express**, this application allows for clean intake and archiving of customer details.

## 🌟 Key Features
- **Client Details**: A streamlined digital intake form for strategic customer overview.
- **Records Archive**: A professional data table (spreadsheet style) to manage and edit past entries.
- **Mobile Responsive**: Fully adaptive design that works on mobiles, tablets, and desktops.
- **Strategic Parameters**: Focused on XR focus, business landscape, and action plan coordination.

## 🚀 Deployment (Recommended for Render)
This project is configured to run as a **Full-Stack** application on platforms like [Render.com](https://render.com).

1. **Build Step**: `npm run build`
2. **Start Command**: `npm start`
3. **Environment Variables**:
   * `NODE_ENV`: `production`

## 🛠️ Local Development
### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Environment
```bash
npm run start-dev
```
This runs the **Vite** frontend on port `5173` and the **Express** backend on port `5000` simultaneously.

## 🤝 Project Structure
- **/src**: React frontend and component logic.
- **/dist**: Production build output (generated on build).
- `server.js`: Express server and MongoDB integration.
- `package.json`: Project configuration and scripts.
